## 📋 Description

Plateforme centralisée de sécurité DevOps qui automatise la détection, l'analyse et la correction des vulnérabilités dans le cycle de développement.

## 🚀 Fonctionnalités

- **Multi-tenant** : Isolation des données par équipe
- **5 types de scanners** : SAST, SCA, Secrets, Conteneurs, IaC
- **Enrichissement** : EPSS, Exploit-DB, CISA KEV
- **Priorisation intelligente** : Score composite basé sur le risque réel
- **Conformité** : SOC2, PCI-DSS, HIPAA, ISO27001, NIST
- **Auto-rémediation** : Correction automatique des secrets et dépendances
- **Dashboard temps réel** : Visualisation des métriques
